# 坦克大战 (Tank Battle)

经典人机对战坦克大战游戏，使用 Python + Pygame 实现。

## 玩法

- 控制黄色坦克消灭所有敌方红色坦克
- 保护你的基地（橙色鹰标）不被摧毁
- 击毁所有敌人即可过关

## 操作

| 按键 | 功能 |
|------|------|
| ↑↓←→ | 移动坦克 |
| 空格 | 发射子弹 |
| ESC | 退出游戏 |
| Enter | 开始/重新开始 |

## 关卡

- 3 个预设关卡，各有不同的墙壁布局
- 每关需要消灭 12 辆敌方坦克
- 敌方坦克分波次出现，场上最多同时存在 3 辆

## 地形

| 图标 | 类型 | 说明 |
|------|------|------|
| 棕色方块 | 砖墙 | 可被子弹摧毁 |
| 灰色方块 | 钢铁 | 不可摧毁 |
| 蓝色方块 | 水域 | 坦克无法通过 |
| 绿色方块 | 树木 | 遮挡视野 |
| 橙色方块 | 基地 | 被击中则游戏结束 |

## 运行

```bash
pip install pygame
python main.py
```

## 技术栈

- Python 3.13
- Pygame 2.6
